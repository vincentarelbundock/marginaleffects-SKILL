
## Extract calling function name from call object {.unnumbered}


### Description

Extract calling function name from call object



### Usage

<pre><code class='language-R'>extract_calling_function(call)
</code></pre>


### Arguments

<table role = "presentation">
<tr><td style = "white-space: nowrap; font-family: monospace; vertical-align: top"><code id="call">call</code></td>
<td>
The call object from which to extract the function name
</td></tr>
</table>


### Value

Character string of the calling function (&quot;comparisons&quot;, &quot;predictions&quot;, or &quot;hypotheses&quot;)
